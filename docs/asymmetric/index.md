# 非对称密码学

!!! 简介

    非对称密码学 也称为公钥密码学


## 主要技术

- RSA
- Curve25519 / Ed25519 / X25519

## Curve25519/Ed25519/X25519

Curve25519/Ed25519/X25519 这是一个家族

> 2005 年，Curve25519 由 `Daniel J. Bernstein`首次发布

Curve25519 用于加解密，X25519用于密钥交换，Curve25519用于签名验证
